# 🏜️ DE-Desert-Emerald | زمردة الصحراء

مشروع متقدم لإدارة وتنظيف وتوحيد البيانات التجارية والطبية المصرية - **منظومة زمردة الصحراء**

---

## 📋 نظرة عامة على المشروع

**زمردة الصحراء** هي منظومة متكاملة لجمع وتنظيف وتوحيد قاعدة بيانات ضخمة تحتوي على **أكثر من 24,000 سجل** من:
- 🏬 المنشآت والمتاجر التجارية
- 🏥 العيادات الطبية والمراكز الصحية
- 💊 الصيدليات والمستشفيات
- 🔬 المختبرات والمركبات الطبية

تم تطويرها لمساعدة المستخدمين في البحث الذكي عن الخدمات التجارية والطبية باستخدام **اللغة المصرية الدارجة** وتحويلها تلقائياً إلى تصنيفات قياسية رسمية.

---

## 🎯 الميزات الرئيسية

### 1️⃣ **البحث الذكي المفهوم للعربية**
- فهم الكلمات المصرية الدارجة (مثل: "عياده"، "دكتور"، "بقاله")
- تحويل تلقائي لتصنيفات مهنية رسمية
- دعم البحث الجغرافي حسب المناطق والمحافظات

### 2️⃣ **تنظيف وتوحيد البيانات**
- تنقية أرقام الهواتف وتصحيحها تلقائياً
- استخراج المناطق الجغرافية من العناوين
- توحيد التصنيفات التجارية والطبية

### 3️⃣ **قاعدة بيانات موحدة**
- جداول منظمة للتجارة والعيادات والصيدليات
- معلومات دقيقة (اسم، عنوان، هاتف، واتساب، تخصص)
- قابلة للنقل لـ PostgreSQL للإنتاج

### 4️⃣ **بنية عمل حديثة**
- **TypeScript** (90.8% من الكود)
- **Node.js** مع Express للـ API
- **SQLite** للتطوير و **PostgreSQL** للإنتاج
- **Drizzle ORM** لإدارة قاعدة البيانات
- **Zod** للتحقق من صحة البيانا��

## 📊 بنية قاعدة البيانات

### الجداول الرئيسية:

| الجدول | الغرض | السجلات |
|---------|------|---------|
| `unified_commercial` | المنشآت التجارية والمتاجر | ~12,000 |
| `unified_clinics` | العيادات الطبية | ~5,000 |
| `unified_pharmacies` | الصيدليات | ~4,000 |
| `unified_hospitals` | المستشفيات | ~2,000 |
| `unified_labs` | المختبرات | ~500 |
| `unified_medical_supplies` | مستودعات المستلزمات الطبية | ~500 |

### الأعمدة الأساسية:
- `id` - المعرّف الفريد
- `name` - اسم المنشأة
- `specialty` / `category` - التخصص أو نوع النشاط
- `address` - العنوان الكامل
- `area` - المنطقة الجغرافية
- `phone` - رقم الهاتف
- `whatsapp` - واتساب (اختياري)
- `doctor` - مدير/صاحب المنشأة
- `classified_by` - الأداة التي نقحت التصنيف

## 🎓 أمثلة متقدمة

### البحث برمجياً:
```python
import sqlite3

DB_PATH = "Titan_Unified_PROD_CLEAN_2026.db"

def search(keyword, area=None):
    conn = sqlite3.connect(DB_PATH)
    cursor = conn.cursor()
    
    query = "SELECT name, specialty, address, phone FROM unified_clinics WHERE 1=1"
    params = []
    
    if keyword:
        query += " AND specialty LIKE ?"
        params.append(f"%{keyword}%")
    
    if area:
        query += " AND area LIKE ?"
        params.append(f"%{area}%")
    
    cursor.execute(query, params)
    return cursor.fetchall()

# الاستخدام
results = search("جلدية", "مصر الجديدة")
for name, spec, addr, phone in results:
    print(f"✓ {name} - {spec} - {addr}")
```
